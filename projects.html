Moody

Improving Breast Cancer Diagnosis

Intel ISEF 2018 #1 Synopsys Silicon Valley Science Fair
TensorFlow Keras Scikit-Learn Jupyter Notebooks

Breast cancer diagnosis is a long and expensive process that isn't available anywhere. In my senior year of high school, my friend and I explored the use cases of machine learning algorithms in breast cancer diagnosis. We evaluated K-NNs, Logistic Regressors, SVMs, and neural networks, long with principal component analysis to find the model that would most accurately predict if someone has a malignant tumor. In the end, we trained a model to be ~98% accurate.
This project won the Synopsys Silicon Valley Science Fair and attended the Intel International Science and Engineering Fair in 2018.

Cows and Bulls

Python JavaScript

This is a computerized version of a word game that I used to play with my family on long road trips. the objective of the game is to guess the CPU's 4-letter word in as few tries as possible. If you get a cow, that means that one of the letters in your guess is in the CPU's word. If you get a bull, that means that one of the letters in your guess is in the CPU's word, and has the same position. Download the game from GitHub and see how good you are!
