Chansey

PennApps XX
React Flask SQL

Chansey is a web application that aims to prevent large-scale outbreaks on university campuses. We ask users to report their symptoms for an in-app currency and give health officials the anonymized health data, so that they can monitor and prevent ilnesses. We also provide users with a notification system that lets them know if the buildings they frequent have a lot of sick people in them.


Moody

HackRU Spring 2019 #1 Overall Best AI Hack
TensorFlow Keras OpenCV Flask Cloud

Moody is a flask application with the goal to help people of all ages with mental health issues. It uses a custom deep convolutional neural network for facial and mood recognition to determine a user's current mood. Moody then logs and tracks the user's mood over time so they can improve their mental health!


Improving Breast Cancer Diagnosis

Intel ISEF 2018 #1 Synopsys Silicon Valley Science Fair
TensorFlow Keras Scikit-Learn Jupyter Notebooks

Breast cancer diagnosis is a long and expensive process that isn't available anywhere. In my senior year of high school, my friend and I explored the use cases of machine learning algorithms in breast cancer diagnosis. We evaluated K-NNs, Logistic Regressors, SVMs, and neural networks, long with principal component analysis to find the model that would most accurately predict if someone has a malignant tumor. In the end, we trained a model to be ~98% accurate.
This project won the Synopsys Silicon Valley Science Fair and attended the Intel International Science and Engineering Fair in 2018.
